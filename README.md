# 🎙️ pleaseeva — OPIc AL 트레이너

**API 비용 0원**으로 매일 OPIc을 실전처럼 연습하고, 내 발화를 분석해 약점과 공부 방향을 제안해 주는 웹앱입니다.

**👉 바로 사용: https://workwwoorrkk12-ctrl.github.io/pleaseeva/**

빌드도 서버도 없는 단일 `index.html` — 모든 처리가 브라우저 안에서 이뤄집니다.

| 기능 | 사용 기술 (전부 무료·브라우저 내장) |
|---|---|
| 질문 읽어주기 (Eva 목소리) | Web Speech API — speechSynthesis (TTS) |
| 내 답변 실시간 영어 전사 | Web Speech API — SpeechRecognition (Chrome 내장) |
| 답변 녹음·다시 듣기 | MediaRecorder |
| 발화 분석·점수·피드백 | 규칙 기반 JavaScript |
| 학습 기록·점수 추이 | localStorage (내 기기에만 저장) |

## 주요 기능

- **실전 모의고사 (14문항)** — 자기소개 + 설문 콤보 2세트 + 돌발 + 롤플레이 + 고난도. 질문 다시 듣기 1회 제한.
- **콤보 집중 연습** — 12개 주제 중 골라 묘사→습관→경험 3문항 훈련.
- **7개 영역 발화 분석** — 답변 분량 / 말하기 속도(WPM) / 어휘 다양성 / 문장 복잡도 / 담화 연결어 / 필러 / 과제 수행.
- **맞춤 학습 플랜** — 약점 기반 "이것부터 공부하세요" 제안 + 예상 등급(IL~AL).
- **성장 기록** — 연속 학습일, 점수 추이 차트, 최근 약점 요약.

## 사용 환경

- **Chrome (데스크톱/안드로이드)** 권장 — 음성인식이 Chrome 내장 기능입니다.
- iOS/Safari 등 음성인식 미지원 환경에서는 답변을 텍스트로 입력하면 동일하게 분석됩니다.
- 마이크 권한을 허용해야 녹음·전사가 동작합니다 (HTTPS 필요 — GitHub Pages는 기본 HTTPS).

## 참고

- 점수와 등급은 학습용 추정치이며, 실제 OPIc 공식 채점과 다를 수 있습니다.
- 학습 기록은 브라우저 localStorage에만 저장됩니다 (외부 전송 없음).
